## 2. 공통 레이아웃

사용자 사이트와 달리 관리자 사이트는 **상단 헤더 + 좌측 사이드바 + 본문** 3단 구조를 사용한다.

| 영역 | 구성 요소 |
|---|---|
| 상단 헤더 | 로고(좌), 관리자명(권한) + 로그아웃 아이콘 + 비밀번호 변경 아이콘 + 홈 아이콘(우) |
| 사이드바 | 단일 메뉴 리스트, 현재 메뉴 활성화, 하위 메뉴는 아코디언 펼침/접기 |
| 본문 | 페이지 제목 + 브레드크럼(홈 > 메뉴명 > 서브메뉴명) + 콘텐츠 영역 |
| 헤더 아이콘 | 로그아웃 → 로그인 페이지 이동 / 비밀번호 변경 → 모달(현재 비밀번호 입력 + 변경할 비밀번호 입력 + 변경할 비밀번호 확인) / 홈 → M1 이동 |

## 4. M1 공사 관리

### M1 공사정보 목록

공사 목록을 표시하는 메인 화면. 검색 조건(공사명/발주업체명/차단업체명 + 기간 + 상태)으로 필터링. 최근 등록순으로 한 페이지에 20개씩 표시. 페이지네이션 적용.

**목록 컬럼**: 번호, 공사명(링크→M1-1), 발주업체명, 차단업체명, 기간, 상태(예정/진행중/완료/취소)

**상태 기준**

| 상태 | 조건 |
|---|---|
| 예정 | 공사 시작일이 오늘보다 미래 |
| 진행중 | 공사 기간 중 |
| 완료 | 공사 종료일이 오늘보다 과거 (정상 종료) |
| 취소 | 발주 취소 처리된 공사 |

### M1-1 공사정보 상세

공사의 기본 정보(공사명, 발주업체, 담당자, 기간, 시간, 도로구분, 길구분, 도로명, 방향, 공사구간, 품목수량, 시연회, 권역, 공사단가)를 표시. 하단 버튼: 수정(M1-4), 목록(M1).

### M1-2 차단정보 목록

해당 공사의 차단 신청 목록. **컬럼**: 번호, 차단업체명(링크→M1-2-1), 차단구간, 신청일, 상태.

### M1-2-1 차단정보 등록·수정

**필드**: 차단업체*(업체찾기 팝업), 차단구간*, TMA1(수량+파일), TMA3(수량+파일), PLCS(수량+파일), 신호수(명수), 의견. 저장 시 경고창 확인 후 목록(M1-2)으로 이동.

### M1-3 진행상황

공사 진행 일정 캘린더. 날짜별 완료/예정/취소 상태 표시. 하단 버튼: 공사정보수정(M1-3-1), 공사금액확인(M1-3-2).

### M1-3-1 공사정보수정

공사 기본 정보 수정 폼. 수정 저장 시 카톡 알림(P08) 발송.

### M1-3-2 공사금액확인

날짜별 품목·수량·단가·금액 상세 테이블. 합계 표시. 엑셀 다운로드 버튼.

### M1-4 공사 등록·수정

**필드**: 공사명*, 발주업체*(업체찾기 팝업 M1-4-1), 담당자*, 기간*(시작일~종료일), 시간*(주간/야간/주야간 + 차단시작시간), 도로구분*(국도/지방도/시도 등), 길구분*(본선/IC/JC 등), 도로명*, 방향*, 공사구간*(시작km~종료km), 품목수량*(TMA1/TMA3/PLCS/신호수), 시연회(날짜+주야간+반공수여부), 권역*, 공사단가*(TMA1/TMA3/PLCS/신호수 각각 입력), 공사없음일(캘린더 다중선택).

등록 저장 시 카톡 알림(P04) 발송.

### M1-4-1 발주업체찾기 (팝업)

검색창에서 발주업체 검색 후 선택. 선택 시 M1-4 폼의 발주업체 필드에 자동 입력.

## 7. M4 기업관리

### M4-1 차단기업 목록

검색(아이디/상호/사업자번호 + 상태). 컬럼: 번호, 상호(링크→M4-1-1), 아이디, 대표자명, 사업자번호, 가입일, 상태(정상/정지). 한 페이지 30개.

### M4-1-1 차단기업 상세보기

기업 정보(상호, 대표자명, 사업자번호, 대표 휴대폰, 이메일, 사업장 주소, 가입일, 상태) + 소속 직원 목록. 하단 버튼: 수정(M4-1-2), 정지/정지해제, 목록.

### M4-1-2 차단기업 등록·수정

**필드**: 상호*, 대표자명*, 사업자번호*(중복확인), 사업자등록증(파일), 대표 휴대폰*, 이메일, 사업장 주소*(우편번호 찾기), 상태*(정상/정지 라디오). 저장 시 경고창 확인.

### M4-2 차단기업직원 목록

검색(아이디/이름 + 소속기업). 컬럼: 번호, 이름(링크→M4-2-1), 아이디, 소속기업, 가입일, 상태. 한 페이지 30개.

### M4-2-1 차단기업직원 상세보기

직원 정보(이름, 아이디, 소속기업, 휴대폰, 이메일, 가입일, 상태) 표시. 하단 버튼: 정지/정지해제, 목록.

### M4-3 발주기업 목록

검색(아이디/상호/사업자번호). 컬럼: 번호, 상호(링크→M4-3-1), 대표자명, 사업자번호, 등록일, 상태. 한 페이지 30개.

### M4-3-1 발주기업 등록·수정

**필드**: 상호*, 대표자명, 사업자번호*(중복확인 — 발주기업 목록 내에서만 중복 체크, 차단기업과는 별도), 사업자등록증(파일), 대표 휴대폰, 이메일, 사업장 주소*(우편번호 찾기), 상태*(공개/비공개 라디오). 저장 시 경고창 확인.

### M4-4 탈퇴·반려회원 목록

검색(아이디/이름(상호)). 컬럼: 번호, 구분(탈퇴/반려), 아이디, 이름(상호), 가입일, 탈퇴(반려)일. 최근 탈퇴순, 한 페이지 30개.

## 9. M6 광고 관리

**필드**: 제목*(30자), 회사명*(30자), 내용*(에디터 — 이미지·유튜브 링크 삽입 가능), 대표이미지*(이미지 파일 1개), 광고기간*(시작일~종료일, 종료일 < 시작일 불가), 사용여부*(공개/비공개). 저장 시 경고창 확인.

## 12. M9 매출 현황

**기준**: 공사 종료일 기준. 진행중 공사는 표시되지 않음.

연월 네비게이션(◀ 2026년 6월 ▶) + 검색(발주업체명/차단업체명) + 엑셀 다운로드. 매출 총계 표시. 컬럼: 번호, 발주업체명(→M4-3-1), 공사명(→M9-1), 차단업체명(→M4-1-1), 기간(시작일~종료일, 실제공사일), 금액. 한 페이지 30개.

### M9-1 매출 상세보기

공사 정보(공사종료일, 공사기간, 공사명, 발주업체명, 총 금액) + 차단기업 정보(아이디, 상호, 사업자번호, 대표자명, 담당자). 날짜별 상세 테이블(날짜, 시간대, 속성, 구분, 품목, 수량, 단가, 금액). 합계 행. 버튼: 엑셀 다운로드, 목록.

**속성 구분**: 시연회(공수/반공수), 공사(일반/긴급), 취소(빈 칸). **구분**: 시연회/공사완료/취소.

## 13. M10 통계

### M10-1 가입탈퇴현황

상단 요약 카드(접속일 기준 차단기업 회원 수, 차단기업직원 회원 수, 등록 발주업체 수 — 탈퇴·반려 제외). 연월 네비게이션 + 엑셀 다운로드. 테이블: 날짜, 가입(차단기업/차단기업직원/발주기업), 탈퇴(차단기업/차단기업직원), 반려(차단기업/차단기업직원). 합계 행 상단 고정.

### M10-2 접속현황

## 17. 퍼블리싱 준비 사항

### CSS 전략

```
roadsafer-dashboard/
├── rss-common.css          ← 공통 CSS (변경 없음)
└── adm/
    ├── adm-common.css      ← 관리자 전용 CSS (컬러 변수 오버라이드)
    ├── adm-sidebar.html    ← 공통 사이드바 컴포넌트
    ├── adm-header.html     ← 공통 헤더 컴포넌트
    └── ath/
        └── parLoginForm.html
```

`adm-common.css`에서 CSS 변수만 덮어씌워 네이비/블루 테마를 적용한다.

```css
/* adm-common.css */
:root {
  --rss-primary:       #3b82f6;
  --rss-primary-dark:  #1d4ed8;
  --rss-primary-light: #dbeafe;
  --rss-sidebar-bg:    #1e2a4a;
  --rss-sidebar-text:  #e2e8f0;
  --rss-sidebar-active:#3b82f6;
}
```

### 사용자 사이트와의 레이아웃 차이점

| 항목 | 사용자 사이트 | 관리자 사이트 |
|---|---|---|
| 상단 헤더 | 없음 | **있음** (로고 + 관리자명 + 아이콘) |
| 사이드바 | 로고 포함 | 헤더 아래 독립 |
| 브레드크럼 | 없음 | **있음** |
| 메뉴 그룹 구분 | 있음 | **없음** (단일 리스트) |
| 컬러 테마 | amber(주황) | **blue(블루)** |
